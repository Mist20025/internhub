## 项目经历

### InternHub — 实习信息管理平台
2026.06 | 个人项目 | [GitHub](https://github.com/Mist20025/internhub)

基于 Spring Boot 3 + MyBatis 开发的 RESTful API 服务，提供实习岗位管理、用户认证和投递记录追踪功能。

**技术栈**：Spring Boot 3.2 · MyBatis 3.0 · MySQL/H2 · JWT · Spring Security · Knife4j · Maven

**主要工作**：
- 采用 Controller → Service → Mapper 三层分层架构设计整个后端系统
- 基于 Spring Security + JWT 实现无状态用户认证，BCrypt 加密存储用户密码
- 设计并实现岗位管理模块（增删改查 + 多条件筛选 + 分页查询），使用 MyBatis 动态 SQL
- 实现投递记录管理功能，支持状态流转（已投递 → 初筛 → 面试 → 录用/拒绝）
- 统一封装 API 响应格式，实现全局异常捕获和参数校验（@Valid）
- 集成 Knife4j（Swagger）自动生成 API 文档，方便前后端联调

**项目亮点**：
- 所有接口均可通过 Swagger 页面在线调试，无需额外工具
- 开发环境使用 H2 内存数据库，无需安装 MySQL 即可运行
- 完整的 README 文档和简历描述，代码结构清晰

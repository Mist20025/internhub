## 项目经历

### InternHub — 实习信息管理平台（功能 + 接口测试）
2026.06 | 个人项目 | [GitHub](https://github.com/Mist20025/internhub)

基于 Spring Boot 3 + MyBatis 开发的 RESTful API 服务，在开发过程中同步进行了系统性测试。

**测试工作**：
- 基于 API 文档（Swagger）设计接口测试用例，覆盖注册、登录鉴权、岗位 CRUD、投递管理等全部 10 个接口
- 使用 Swagger 调试工具和 Postman 执行接口测试，验证请求参数校验、响应状态码和返回数据格式
- 针对 JWT 认证场景设计了安全性测试：验证无 Token 访问返回 403、错误 Token 返回异常、过期 Token 处理
- 测试分页查询边界条件：page=0、超大 pageSize、空关键词、多条件组合筛选
- 编写测试记录，记录缺陷和接口行为偏差

**测试成果**：
- 发现并修复了 Swagger 文档页面因安全配置导致的 403 问题
- 验证了 POST 请求参数校验（中文编码兼容性）
- 确认了 H2 内存数据库环境下所有 SQL 查询的正确性
